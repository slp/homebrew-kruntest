# Slp Kruntest

## How do I install these formulae?

`brew install slp/kruntest/<formula>`

Or `brew tap slp/kruntest` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "slp/kruntest"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
